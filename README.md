# Privacy-Armor
## 🚀 AI Privacy Shield for Real-Time Data Protection

An intelligent system that acts as a privacy firewall for text data by detecting and masking sensitive information in real time. Powered by NLP and pattern recognition, it ensures zero data leakage across AI systems, chat platforms, and digital environments.

Built for the future of secure AI.

## 🔥 Advanced Features

- Uses spaCy NLP for real-time PII detection
- Hybrid system: NER + Regex
- Evaluation based on Precision, Recall, F1 Score
- Supports multi-step agent interaction

## 📊 Metrics

| Metric    | Description |
|----------|------------|
| Precision | Correct redactions / total redactions |
| Recall    | Correct redactions / total PII |
| F1 Score  | Harmonic mean (final reward) |
